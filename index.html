Proyectos

Ratio Memes

Disponible para Android e iOS en Google Play y App Store, también cuenta con una página web introductoria. Diseñada para ver, buscar y disfrutar de tus memes favoritos sin ocupar espacio en tu dispositivo, utiliza la nube de Firebase para almacenar los archivos de manera eficiente. La app fue desarrollada en Kotlin con Jetpack Compose para Android y en SwiftUI para iOS. Con funciones que permiten compartir memes como audio, imagen o video, integrarse fácilmente con Twitter, y agregar memes a tus favoritos, Ratio Memes sigue recibiendo actualizaciones para mejorar la experiencia.

CNC Laser

Disponible para Android e iOS, diseñada para optimizar el trabajo con maquinaria CNC. Con esta herramienta, los usuarios pueden calcular costos, estimar tiempos de trabajo, obtener vectores y diseños personalizados, y generar comprobantes. La app está desarrollada en lenguajes nativos para garantizar el mejor rendimiento: Kotlin con Jetpack Compose para Android y SwiftUI para iOS. Además, integra bases de datos locales para mayor eficiencia y Firebase para sincronización online. CNC Laser ya cuenta con más de 20.000 descargas en la Play Store y App Store.
